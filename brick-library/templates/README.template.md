All entity endpoints follow RESTful conventions:
- `GET /api/{entity}` - List all
- `GET /api/{entity}/:id` - Get by ID
- `POST /api/{entity}` - Create
- `PUT /api/{entity}/:id` - Update
- `DELETE /api/{entity}/:id` - Delete
