## Severity Guide

- **Critical**: blocks release/demo, data loss, or complete flow failure.
- **High**: major feature broken but workaround exists.
- **Medium**: partial behavior mismatch, non-blocking.
- **Low**: cosmetic/minor validation or message issue.

## Status Guide

- **Open** -> **In Progress** -> **Fixed** -> **Retest Failed/Retest Passed** -> **Closed**
