# STANDARD INVOICE ENTITIES (ALWAYS INCLUDE WHEN INVOICE IS ENABLED):

## 1. customers (module: "shared" if used by other modules)
Required fields:
- `company_name` (string, required)
- `email` (string, required)
- `phone` (string)
- `address` (text)
- `vat_number` (string)

## 2. invoices
Required fields:
- `invoice_number` (string, required, unique)
- `customer_id` (reference to `customers`, required)
- `issue_date` (date, required)
- `due_date` (date, required)
- `status` (string, options: ["Draft", "Sent", "Paid", "Overdue"], required)
- `subtotal` (decimal)
- `tax_total` (decimal)
- `grand_total` (decimal)

Set `features.print_invoice: true`
Set `children` config:
```
"children": [{
  "entity": "invoice_items",
  "foreign_key": "invoice_id",
  "label": "Line Items",
  "columns": ["description", "quantity", "unit_price", "line_total"]
}]
```

## 3. invoice_items
Required fields:
- `invoice_id` (reference to `invoices`, required)
- `description` (string, required)
- `quantity` (decimal, required)
- `unit_price` (decimal, required)
- `line_total` (decimal)

# ENTITY FORMAT (STRICT):
Each entity object MUST contain:
- `slug` (string, plural snake_case like `invoices`, `customers`)
- `module` (string) - "invoice" for invoice entities, or "shared" if used by other modules
- `fields` (array of field objects)

Optional entity properties:
- `display_name` (string) - Human-readable name
- `display_field` (string) - Field to show when referenced
- `ui` (object): `{ search, csv_import, csv_export, print }`
- `features` (object): `{ print_invoice: true/false }`
- `children` (array) - For parent-child relationships like invoice -> line items

# FIELD FORMAT (STRICT):
Each field MUST contain:
- `name` (string)
- `type` (string): string, text, integer, decimal, boolean, date, datetime, reference

Optional field properties:
- `label` (string)
- `required` (boolean)
- `reference_entity` (string) - For type "reference"
- `options` (array) - For selectable fields (use type: "string" with options, NOT type: "enum")
- `unique` (boolean)

# SYSTEM FIELDS - DO NOT INCLUDE:
- `id`, `created_at`, `updated_at` - These are auto-generated
